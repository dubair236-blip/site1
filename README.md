# 🦆 Xalyava Stars - Игровое WebApp приложение

Веб-приложение для Telegram бота @XalyavaStars_RoBot с мини-играми на звёзды Telegram.

## 🎮 Игры

### ❌⭕ Крестики-Нолики
- Онлайн игра против друзей
- Автоматический запуск с ботом через 30 секунд
- Ставки от 1⭐

### 🎲 Кубик
- Брось 3 кубика против соперника
- Победитель забирает банк
- Онлайн и оффлайн режимы

### 🥤 Найди шарик
- Угадай под каким стаканом шарик
- Выигрыш x1.5 от ставки
- Оффлайн игра

Откройте бота @XalyavaStars_RoBot и нажмите кнопку "🎮 Игры"
